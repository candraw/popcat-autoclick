# popthe-popcat
Aplikasi Otomasi Klik situs popcat.click.
Jalankan dan biarkan aplikasi ini bekerja

Beberapa yang diperlukan untuk menjalankan aplikasi diantaranya : 
- Google chrome
- Pyhton Installed
- Selenium library 
- Chrome web driver 
- Text Editor 

NB : Pastikan versi Google Chrome dan WebDriver sama
